# **App Name**: Seu Espaço Web

## Core Features:

- Vitrine de Serviços: Apresentar visualmente os serviços de limpeza com galerias de imagens.
- Orçamento Instantâneo: Gerar orçamentos de serviços com base na metragem quadrada ou no número de cômodos, usando uma ferramenta de IA para oferecer estimativas personalizadas.
- Agendamento de Serviços: Permitir agendamentos de serviços com base no orçamento.
- Integração com WhatsApp: Facilitar consultas diretas de clientes via WhatsApp.

## Style Guidelines:

- Cor primária: Azul forte (#004AAD), transmitindo confiança na marca. Complementando o design existente. O matiz é próximo ao do ultramarino.
- Cor de fundo: Cinza claro (#F4F9FC), usado para dar um cenário limpo e fresco para o conteúdo.
- Cor de destaque: Teal (#00C07F), fornece uma cor secundária vibrante, alinhada com limpeza, ação e modernidade. Este destaque contrasta com o fundo claro.
- Fonte: 'PT Sans', sans-serif, fornecendo uma tipografia moderna e legível para manchetes e corpo do texto. O prompt não sugere um estilo visual que exija várias fontes.
- Usar ícones de linha simples para serviços e benefícios.
- Layout limpo e espaçoso para mostrar serviços e informações de forma eficaz.
- Animações de transição sutis em estados de foco para melhorar a interatividade.
